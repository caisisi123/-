# 简介
 SPA项目，主要使用了Vue3框架、Element Plus UI和Echarts。该项目的功能是记录每天的工作计划和完成情况，并能够汇总查看过去的计划情况。此外，它还提供了番茄钟和正计时、对记录和专注统计的功能。对页面加载做过许多优化，所以使用体验尚可哈哈。


## 功能:
* 离线储存

* 记录每日计划和标记是否完成

* 查看历史计划，也可以查看过去一段时间的计划清单

* 番茄钟和正计时

* 统计专注情况和计划情况
